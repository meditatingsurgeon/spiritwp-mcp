# SpiritMCP Bridge

A dual-mode WordPress plugin that bridges The Spiritual Agency's MCP server with self-hosted WordPress sites.

**Mode 1 — Inbound (MCP → WP):** Exposes a authenticated REST endpoint so the MCP server can read and write WordPress data (posts, meta, users, options) on behalf of connected sites.

**Mode 2 — Outbound (WP → MCP):** Registers WP-CLI and admin-triggered actions that push site events and content changes back to the MCP server for centralised logging and automation.
